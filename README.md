# Chat-App

## [Click here to view Chat-App](https://chat-application-g7a9.onrender.com)

### Setup `.env` File in the Root Directory

```plaintext
PORT=...
MONGO_DB_URI=...
JWT_SECRET=...
NODE_ENV=...

```

### Build the app

```shell
npm run build
```

### Start the app in root directory

```shell
npm start
```
